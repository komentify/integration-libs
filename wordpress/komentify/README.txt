Allows you to easily integrate komentify into your wordpress app to engage your users.

== Description ==

This plugin provides support for wordpress. Only define the
APP Id of your website and your wordpress blog is ready to go.

This plugin will embed an iframe that calls the komentify server to display the UI of your comment widget.
You have full control over your comments by logging in to komentify and going to the dashboard.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Komentify screen to configure the plugin

Almost every business owns a website these days. However, few of them are able to engage their visitors. We provide a solution. A comment widget with a strong focus on customizability, privacy and a modern look.
